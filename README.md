# Sistema de Inventario PHP

Sistema web básico para gestión de inventario desarrollado en PHP y MySQL.

## Tecnologías
- PHP
- MySQL
- HTML
- CSS

## Funciones
- Agregar productos
- Editar productos
- Eliminar productos
- Listar inventario
- Post de ventas
- Modulo de compras
- Modulo de finanzas
